  3. Multivariate Linear Regression
  4. Outputting Regression Results
  5. Fixed Effects
  6. Instrumental Variables (IV)
  7. Data Wrangling
  8. Differences in Differences (DiD)
  9. Regression Discontinuity Design (RDD)
